INSTRUCTIONS ON USING THE USB SOFTWARE COPIER

1. Use the dropdown menu to select which software you would like to be copied.
2. Click the 'SoloX' checkbox to also include a copy of the SoloX (UEIPAC Option 11/12) software also.
3. Click the 'Begin File Transfer' button in the center. 
4. Another window will open letting the user know which drives the software will be copied to. Please make sure there are the expected number of USBs/drives.
5. Click 'Okay' to start the copying process.
6. Once the software finishes copying, another window will appear and let the user know it has been completed.
